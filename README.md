# Modular-Power-Bank
A power bank with attachable modules and up to 45W output
# Features
- USB PD output up to 45W (may handle 60W)
- 0.91 inch OLED for status monitoring
- Attachable modules
- Repairable design
# Modules
- LED controller - Allows you to control 5V LEDs with Home Assistant
I also plan to make a LiPo charger module that will support charging up to 3S LiPos.
Non of the PCBs are fully tested so I can't guarantee that they will work correctly
